## About Me

AI/ML Engineer focused on retrieval-augmented generation, knowledge graphs, and applied ML systems that go from notebook to production. I care about retrieval that's actually precise, models that ship with monitoring attached, and code that's readable six months later.

- 🔍 Building **RAG evaluation frameworks** — measuring retrieval quality instead of guessing at it
- 🕸️ Exploring **GraphRAG** — combining vector search with knowledge-graph reasoning
- ⚙️ Shipping **production ML** — XGBoost/Optuna tuning, MLflow tracking, FastAPI serving, Docker, CI/CD, drift monitoring
